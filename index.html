🚀 快速安装

# 1. 克隆仓库
git clone https://github.com/chaoliuzhu65-tech/universal-travel-planner-skill.git

# 2. 复制到 WorkBuddy Skills 目录
cp -r universal-travel-planner-skill ~/.workbuddy/skills/

# 3. 配置环境变量（可选）
export AMAP_WEB_KEY=你的高德Web服务Key

# 4. 在对话中使用
# 说 "帮我规划下周去北京的出差行程" 即可触发
